# Book catalogue

Computer program in Java 25 + Maven + Swing for managing the catalog of books.

## Requirements
- Java 25 (JDK 25)
- Maven 3.9+
- IntelliJ IDEA (recommended)

## Run
mvn clean compile

## Functionality
- Adding/deleting/updating books
- Search by name
- Saving/loading the directory to a file (serialization)
